## Tentang Proyek

Proyek **Modelling Geofisika** menyediakan platform untuk melakukan simulasi dan visualisasi data geofisika secara interaktif.  
Dengan dukungan **Python** dan **Jupyter Notebook**, proyek ini memungkinkan pengguna untuk:
- Mensimulasikan dinamika gelombang seismik.
- Menganalisis data gravitasi dan metode elektromagnetik.
- Mengeksplorasi data melalui visualisasi grafik dan animasi.
