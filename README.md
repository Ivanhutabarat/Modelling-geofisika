## 🚀 Tentang Proyek  

Proyek **Modelling Geofisika** menyediakan platform open-source untuk **simulasi dan visualisasi data geofisika** secara interaktif.  
Dengan dukungan **Python**, **Jupyter Notebook**, dan teknologi pendukung lainnya, proyek ini memungkinkan pengguna untuk:
✅ **Simulasi Gelombang Seismik** dengan metode numerik.  
✅ **Analisis Data Gravitasi** untuk pemetaan bawah permukaan.  
✅ **Pemodelan Elektromagnetik** untuk eksplorasi geologi.  
✅ **Visualisasi Interaktif** menggunakan Matplotlib dan Plotly.

## 🛠️ Cara Instalasi  

Pastikan kamu telah menginstal **Python 3.8+** dan semua dependensi yang diperlukan. Lakukan langkah-langkah berikut:

```bash
git clone https://github.com/Ivanhutabarat/Modelling-geofisika.git
cd Modelling-geofisika
pip install -r requirements.txt


```

Jika kamu ingin menjalankan proyek melalui **Jupyter Notebook**, pastikan untuk menginstalnya juga:

```bash
pip install jupyter
```

---

## 📜 Penggunaan  

### 🔹 Melalui Terminal  

Jalankan simulasi sederhana dengan perintah:

```bash
python run_simulation.py --duration 10 --resolution 100
```

### 🔹 Melalui Jupyter Notebook  

Kamu bisa mulai dengan:

```bash
jupyter notebook
```

Kemudian buka file `Modelling-geofisika.ipynb` untuk eksplorasi interaktif.
